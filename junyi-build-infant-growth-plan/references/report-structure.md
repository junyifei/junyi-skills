# 0—3 岁报告结构

最终 Markdown 按以下顺序：

1. `# 0—3 岁全年成长规划`
2. `## 使用说明与边界`
3. `## 宝宝与家庭情境`
4. `## 证据地图`
5. `## 优势、兴趣与关系资源`
6. `## 日常参与画像`
7. `## 当前支持重点`
8. `## 90 天家庭行动实验`
9. `## 全年路线图`
10. `### 第一季度`、`### 第二季度`、`### 第三季度`、`### 第四季度`
11. `## 照护者协作与家庭负担`
12. `## 复盘安排`
13. `## 需要进一步了解`
14. `## 专业支持提示`

证据地图至少包含：编号、观察事实、来源、日期或时间范围、情境、解释或假设、置信度。

证据地图必须逐项忠实继承 intake；不得补写未知时间、改换来源、删掉冲突或把未知项写成结论。正文中的每条优势注明 `证据强度：高/中/低（待复现）`，单次观察不得写成稳定特质。

每项实验用 `### 实验：名称`，并写目标、依据、成人与环境行动、频率、成功信号、停止或调整条件、记录方式。

“成功信号”前先写 `基线` 和 `复盘时点`，确保比较的是同一情境、相近任务与相同口径。需要家庭先确认的资源或安排必须写 `启动前提：待家庭确认`。

四个季度都要写方向、月度观察锚点和进入下一季度前的决策条件；不得写成临床里程碑倒计时。
