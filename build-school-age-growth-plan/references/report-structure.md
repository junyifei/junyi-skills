# 6—12 岁报告结构

最终 Markdown 按以下顺序：

1. `# 6—12 岁全年成长规划`
2. `## 使用说明与边界`
3. `## 孩子、家庭与学校情境`
4. `## 孩子的声音`
5. `## 证据地图`
6. `## 优势、兴趣与关系资源`
7. `## 学习与探究过程画像`
8. `## 自我管理、生活节律与数字环境`
9. `## 情绪、同伴与家庭关系`
10. `## 当前支持重点`
11. `## 90 天共同行动实验`
12. `## 全年路线图`
13. `### 第一季度`、`### 第二季度`、`### 第三季度`、`### 第四季度`
14. `## 家庭与学校协作`
15. `## 复盘安排`
16. `## 需要进一步了解`
17. `## 专业支持提示`

证据地图至少包含编号、观察事实、来源、日期或时间范围、情境、解释或假设、置信度。

证据地图必须逐项忠实继承 intake；不得补写未知时间、改换来源、删掉冲突或把未知项写成结论。正文中的每条优势注明 `证据强度：高/中/低（待复现）`，单次观察不得写成稳定特质。

每项实验用 `### 实验：名称`，依次写目标、依据、孩子行动、成人与环境行动、频率、成功信号、停止或调整条件、记录方式。

“成功信号”前先写 `基线` 和 `复盘时点`，确保比较的是同一情境、相近任务与相同口径。需要孩子、家庭或学校先确认的事项必须写 `启动前提：待确认`。章节标题中的“共同”仅指协作范围；没有与该实验绑定的孩子选择证据时，不得写成孩子已经共同设计。

四个季度都写方向、月度观察锚点和调整条件；不得写分数保证、名次、升学或录取预测。
