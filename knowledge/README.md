# Knowledge

这里存放已经有价值、但尚未满足正式 Skill 质量门的知识卡。

当前公开目录首先说明晋升规则，不用知识卡数量制造成熟感。新的知识卡只有在完成隐私、权利和可移植性检查后才会加入；本地私有知识库不属于公开仓库。

一条知识只有在满足以下条件后，才晋升为独立 Skill：

1. 有清楚的触发任务和不适用边界。
2. 有可重复执行的步骤，而不只是观点。
3. 知道需要哪些输入证据、会产出什么结果。
4. 至少有真实任务记录，包含好结果、差结果或人工修改。
5. 已完成隐私、权利、路径和外部依赖检查。
6. 能设计验收标准，判断它何时算完成、何时必须失败或请用户补充。

推荐每张知识卡至少记录：一句话结论、适用场景、反例、证据来源、验证次数、当前状态和下一次要验证的问题。

更完整的证据等级、成熟度和公开边界见 [`guide/METHOD-AND-EVIDENCE.md`](../guide/METHOD-AND-EVIDENCE.md)。
