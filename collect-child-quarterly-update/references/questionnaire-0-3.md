# 0—35 月龄季度补充问卷

只在季度开始日完整月龄为 0—35 时读取。最小版资料已充分时无需继续询问。

## 1. 关系与沟通

- 最近一次孩子主动寻求熟悉成人关注、安慰或共同参与，发生了什么？
- 孩子现在主要通过声音、表情、手势、动作或语言表达什么？与上季度相比有什么具体变化？

## 2. 游戏、探索与动作

- 孩子最近反复进行什么游戏或探索？请描述一次主动开始、重复或改变玩法的事件。
- 在移动、户外或使用双手时，最近出现过什么新事件？单次事件可以记录，但不要直接写成已经稳定掌握。

## 3. 情绪、分离与转换

- 分离、重逢、受挫、害怕或兴奋时，孩子最近一次怎样表现？成人怎样回应，什么有帮助？
- 入睡、出门、回家、洗澡、进餐等转换中，哪个场景最顺，哪个最容易冲突？

## 4. 进食、睡眠和日常参与

- 过去一个月的进食、睡眠、午睡和夜醒有什么变化？只描述观察，不自行解释医学原因。
- 在穿脱、清洁、喝水、收纳或如厕中，孩子最近愿意参与哪些步骤？

## 5. 照护环境与多来源观察

- 主要照护者、托育安排、家庭语言或生活环境有什么变化？
- 其他照护者或托育人员观察到什么？与家长观察一致或不同在哪里？

其他照护者或托育人员视角是建议项，不是硬门槛。没有其他照护者时明确记录“不适用”。
