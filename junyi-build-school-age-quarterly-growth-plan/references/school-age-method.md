# 72—144 月龄季度方法

## 1. 方法立场

- 孩子是计划参与者，不是被执行对象；
- 学习过程优先于分数和排名；
- 自我管理需要适龄支架，不等于成人完全退出；
- 兴趣是否值得投入，看持续行动而非一时兴奋；
- 冲突和失败后的恢复过程比消除所有困难更值得观察。

## 2. 五维度画像

1. **家庭教育目标**：核心追求、底线，以及口头目标和实际投入是否一致。
2. **孩子优势与挑战**：2—3 个有具体场景的优势，1—2 个真正需要投入的当前门槛。
3. **陪伴者风格**：谁陪、怎么陪、实际能投入多少，区分理想投入与真实投入。
4. **现有资源**：时间、预算、活动、环境、工具、社交与家庭支持；活动过多时提示做减法。
5. **本季新数据**：最近三个月的转折、动力窗口、持续卡点和环境变化。

稳定优势与新兴优势要区分；一次性事件可以记录，趋势判断需要跨时间或多来源证据。

## 3. 六个真实领域

孩子本人反馈、学习过程、自我管理与生活节律、情绪与同伴关系、兴趣创作与贡献、学校与环境变化。这些领域用于寻找候选方向，不要求每季度全部展开。

## 4. 分龄参与

### 72—107 月龄

- 使用“最喜欢、最烦、最难、最想试”等具体问题；
- 一次只讨论一个真实场景；
- 允许口头、画图、排序或选择卡表达；
- 不要求孩子系统理解整份计划。

### 108—144 月龄

- 邀请孩子说明想保留、停止和尝试什么及理由；
- 共同确认成功画面与最低版本；
- 允许孩子提出替代方案和复盘方式；
- 不把成人预先决定的方案包装成共同制定。

## 5. 行动与学科专项

每项行动具体到场景、第一步、所需时间、负责人和退出方式。优先使用低成本试验、可见流程、选择权、角色分工和短周期复盘。

仅当主攻确实涉及学科时，增加完整学习方案：

1. 当前水平与置信度；
2. 已核验材料或类别建议；
3. 一次学习怎么做及家长话术；
4. 分阶段进阶路线与可观察标志；
5. 课程或服务的选择标准；
6. 评估建议与调整条件。

学习方向必须同时保护睡眠、运动、关系和非功利兴趣。
