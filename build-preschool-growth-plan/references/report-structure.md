# 3—6 岁报告结构

最终 Markdown 按以下顺序：

1. `# 3—6 岁全年成长规划`
2. `## 使用说明与边界`
3. `## 孩子、家庭与学前环境`
4. `## 证据地图`
5. `## 优势、兴趣与关系资源`
6. `## 游戏、语言与探究画像`
7. `## 动作、生活自主与日常节律`
8. `## 情绪、同伴与集体参与`
9. `## 当前支持重点`
10. `## 90 天家庭与学前行动实验`
11. `## 全年路线图`
12. `### 第一季度`、`### 第二季度`、`### 第三季度`、`### 第四季度`
13. `## 入学过渡与环境协作`
14. `## 复盘安排`
15. `## 需要进一步了解`
16. `## 专业支持提示`

证据地图至少包含编号、观察事实、来源、日期或时间范围、情境、解释或假设、置信度。

证据地图必须逐项忠实继承 intake；不得补写未知时间、改换来源、删掉冲突或把未知项写成结论。正文中的每条优势注明 `证据强度：高/中/低（待复现）`，单次观察不得写成稳定特质。

每项实验用 `### 实验：名称`，依次写目标、依据、孩子/成人/环境行动、频率、成功信号、停止或调整条件、记录方式。

“成功信号”前先写 `基线` 和 `复盘时点`，确保比较的是同一情境、相近任务与相同口径。需要孩子、家庭或园所先确认的资源与安排必须写 `启动前提：待确认`。孩子未实际参与选择时，孩子行动写成待确认提案，不得声称已经共同设计。

四个季度都写方向、月度观察锚点和调整条件；入学过渡不得生成评分或录取预测。
