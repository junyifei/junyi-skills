# 72—144 月龄主攻方向决策方法

## 1. 先排除

- 未处理的即时安全风险；
- 需要专业判断的医学、心理、发育或教育诉求；
- 缺少有效全年底座、季度开始月龄或 `status: ready` 的季度更新；
- 孩子直接反馈未达到分龄要求；
- 目标依赖尚未确认的课程、学校、预算或第三方承诺。

## 2. 形成候选方向

候选只能来自全年支持重点、本季 `Qxx`、孩子直接表达、家庭真实目标和已确认环境变化。每个候选至少绑定一条证据。

## 3. 透明比较

对每个候选判断：

1. **证据强度**：是否跨时间、跨场景或多来源；
2. **时间窗口**：现在做是否明显优于以后做；
3. **孩子动力与压力**：孩子愿不愿参与，当前意义感和压力怎样；
4. **执行可行性**：成人、环境、任务拆分和支架能否在 90 天内改变；
5. **投入回报**：预期收益是否匹配孩子与家庭负担；
6. **家庭底线**：是否损害睡眠、运动、关系或非功利兴趣。

不机械打分。选 1—2 个主攻、1—2 个维持方向。超过两个主攻时，删除证据最弱、窗口最不紧迫或执行性最低的方向。没有明显紧迫方向时，顺着孩子当前最强且持续的动力。

## 4. 主攻与维持的区别

- 主攻：需要新增结构、稳定提醒或专项投入；
- 维持：现有优势不需额外加码，只需继续提供材料和场景；
- 每个维持方向写明什么信号出现时才升级为主攻；
- 每份报告说明为什么选择当前方向、为什么其他方向本季度暂不做。

## 5. 负担与容错

- 每周新增或需要持续提醒的核心行动最多 4 个；
- 已稳定执行的阅读、运动、创作和家庭活动写为维持；
- 开学前作息恢复等短期安排写条件性阶段任务；
- 没有孩子确认时，第 0 步先确认而不是直接执行；
- 每项行动都有忙碌周最低版本和孩子可提出的调整方式；
- 两周无变化先检查难度、环境、时间、支架和记录口径；
- 压力、睡眠、关系或身体信号恶化时暂停并重新判断。
