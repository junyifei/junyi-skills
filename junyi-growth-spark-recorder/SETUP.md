# junyi-growth-spark-recorder · 手把手安装（小白版）

跟着做，5 分钟装好。**每一步只做一件事，做完再做下一步。**

---

## 1. 这个 Skill 帮你解决什么问题

一句话：**遇到「孩子有个闪光瞬间，你觉得挺好，但说不清到底好在哪、也不知道下次怎么做更好」的时候，用它。**

它会帮你：把那一刻记下来 → 从孩子发展的角度讲清楚这意味着什么 → 再给你一条下次能直接照做的建议，并自动存档。

你只管发一句你看到的，剩下的它来。

---

## 2. 装之前先确认（环境检查）

你需要：

- 一台电脑，已经装好 **Claude Code**
- Claude Code 能正常跟你打字对话

⚠️ **先自检一下**：在 Claude Code 里输入这句——

> 帮我确认 Claude Code 能正常运行

它能正常回你话，就继续往下。如果回不了或报错，先把 Claude Code 弄好，再回来装。

---

## 3. 三步装好

### 第一步：把文件夹放进 skills 目录

把 `junyi-growth-spark-recorder` 这**整个文件夹**，放到下面这个位置：

- ⚠️ **Mac**：`~/.claude/skills/`
- ⚠️ **Windows**：`C:\Users\你的用户名\.claude\skills\`

不会手动放？把这句直接发给 Claude Code，让它帮你放：

> 帮我把 junyi-growth-spark-recorder 这个文件夹复制到我的 Claude Code skills 目录

> 用 OpenClaw / Codex / Hermes 的同理，对应目录见仓库 README 的「怎么安装」。

### 第二步：要不要改成你自己的信息？

这个 skill 已经**去个人化、开箱即用**，绝大多数人**无需替换**，可以直接跳到第三步。

只有一个**可选**项——想让分析更贴合你家娃，可以在 `SKILL.md` 的「配置」区登记孩子信息：

| 要改的地方 | 改成什么 | 必须填吗 |
|---|---|---|
| `SKILL.md`「配置」区 `孩子A` 那一行 | 你孩子的名字 + 出生年月 | 可选（不填也能用，它会从你说的话里自动认名字） |
| `孩子B` 那一行 | 第二个孩子；只有一个娃就删掉这行 | 可选 |
| 归档目录 `growth-records/` | 想把记录存到别的地方，就改成你的路径 | 可选（默认就好） |

⚠️ 嫌麻烦就**整段跳过**，直接用，完全不影响。

### 第三步：重启 + 验证装好没

⚠️ **新开一个 Claude Code 会话**（它只在启动时扫描 skills，不新开会话发现不了）。

然后输入这句**触发话术**试试：

> 记录一下，孩子今天主动把玩具分给了小朋友

如果它开始分三层回你——**📸 记录 ／ 🔬 发展分析 ／ 🧠 模型复盘**——就说明装好了。🎉

## 4. 用一次试试

复制下面这句，发给 Claude Code：

> 帮我复盘：今天孩子在超市自己算出了一袋糍粑多少钱

你会看到它：把这件事记下来 → 分析这在孩子这个年龄意味着什么 → 用一个思维模型给你「下次可以试」的具体做法。

看到这三层，就算真正会用了。
