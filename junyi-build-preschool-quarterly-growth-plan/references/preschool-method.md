# 36—71 月龄季度方法

## 1. 方法立场

- 游戏是理解兴趣、表达、规则与关系的重要场景；
- 有限选择比连续命令更能支持参与；
- 可见步骤和环境调整比抽象说教更容易执行；
- 生活自主是参与过程，不是越快独立越好；
- 孩子直接表达必须进入季度决策。

不使用月龄表给孩子评分，不把单次合作、拒绝、冲突或作品表现写成稳定人格和能力。

## 2. 五维度画像

1. **家庭教育目标**：用 1—2 句话提炼核心追求与底线。
2. **孩子优势与成长中课题**：优势 2—3 条、课题 1—2 条，全部使用具体场景。
3. **陪伴者风格**：谁主要陪伴、各自擅长什么、实际能投入多少时间。
4. **现有资源**：学校、时间、活动、环境、社交与家庭支持。
5. **本季新数据**：最近的变化、新兴趣、新挑战、突破和卡点。

资料冲突时并列呈现事实和不同观点。决策先看孩子实际状态，再看成人实际行为，最后看口头目标；正文温和指出张力，不指责。

## 3. 年龄内分流

### 3—4 岁

最接近婴幼儿方法。重点是观察、环境提供、情绪命名、共同游戏和简单参与，不要求稳定完成。

### 4—5 岁

可以设置轻量游戏目标，支持更长的共同游戏、讲述、合作、规则理解和生活步骤，但保留选择与退出空间。

### 5—6 岁

可以更明确地支持生活自主、合作、表达和对新环境的准备。入学准备以节律、沟通、身体、关系和任务理解为主，不以识字量、算术量或固定学习时长为季度目标。

## 4. 游戏化建议原则

优先使用：

- 角色游戏和彩排；
- 两个以内的真实选择；
- 图片、实物或两三步可见流程；
- 共同收尾、保存未完成作品；
- 家庭真实任务；
- 成人示范表达，再逐步减少支架。

每项建议写清适用场景、怎么玩、成人怎样回应、孩子怎样参与或拒绝、建议频率和要观察的信号。建议是菜单，不是打卡清单。
