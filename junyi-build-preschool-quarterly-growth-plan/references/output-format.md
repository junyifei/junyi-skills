# 36—71 月龄季度计划输出格式

标题：`# {匿名代号} {YYYY} Q{N} 学前儿童个性化发展计划`

基础计划通常约 4000—7000 个中文字符。结构：

1. `## 使用说明与边界`
2. `## 家庭画像速览`
3. `## 五维度画像提炼`
4. `## 本季度聚焦方向`
5. `### 决策透明：为什么选择这些方向`
6. `### 聚焦方向`
7. `### 自然维持方向`
8. `## 游戏化学习与生活建议`
9. `## 重要陪伴者专属互动方案`（有可靠资料时）
10. `## 家长话术`
11. `## 避坑提醒`
12. `## 观察与复盘`
13. `### 绿灯：继续保持的信号`
14. `### 黄灯：需要微调的信号`
15. `### 红灯：停止或求助信号`
16. `## 每月复盘三问`
17. `## 下季度展望`

## 内容约束

- 文档头部写周期、季度开始与结束月龄、家庭策略和制定日期。
- 家庭画像与五维度提炼中的关键事实全部绑定 `Exx` 或 `Qxx`。
- 使用 `#### 聚焦：名称` 1—2 个、`#### 维持：名称` 1—2 个。每个聚焦方向写证据、孩子直接表达、为什么现在、三个月后的可观察画面和为何其他方向暂不加码。
- 每个聚焦方向在“游戏化学习与生活建议”中提供 3—5 个建议，使用 `### 方向：名称` 分组，并以 `#### 游戏建议：名称` 标记每个建议。每个建议包含场景、玩法、成人回应、孩子选择、频率、最低版本和观察信号。
- 建议根据季度开始月龄使用 3—4、4—5 或 5—6 岁强度，不把学前准备写成学科达标。
- 专属互动仅基于真实家庭角色、时间和优势；不替他人承诺。
- 家长话术使用“场景、可以说、避免说”三列，至少 5 个场景；涉及接送与陪伴时只能使用已确认安排。
- 避坑提醒写 3—5 条，每条包含本家庭风险、原因和替代做法。
- 绿灯、黄灯、红灯描述相对本计划的变化与求助边界，不作诊断或结果保证。
- 每月三问分别检查孩子体验、实际效果和家庭负担。
- 下季度展望只作条件性预判。
