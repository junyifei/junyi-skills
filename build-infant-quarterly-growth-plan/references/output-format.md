# 0—35 月龄季度计划输出格式

标题：`# {匿名代号} {YYYY} Q{N} 婴幼儿个性化发展计划`

基础计划通常约 4000—7000 个中文字符。资料丰富时可以展开，但不得重复罗列材料。

## 固定结构

1. `## 使用说明与边界`
2. `## 家庭画像速览`
3. `## 五维度画像提炼`
4. `## 本季度发展观察重点`
5. `## 环境提供方案`
6. `## 重要陪伴者专属互动方案`（有可靠资料时）
7. `## 本计划观察指标`
8. `## 家长话术`
9. `## 避坑提醒`
10. `## 每月复盘三问`
11. `## 下季度展望`

## 内容要求

- 文档头部写计划周期、季度开始月龄、季度结束月龄、一句话家庭策略和制定日期。
- 家庭画像覆盖教育目标、孩子此刻的样子、陪伴模式、现实限制、现有资源和季度事件；只写有 `Exx` 或 `Qxx` 支持的内容。
- 五维度画像依次写家庭目标、孩子优势与成长中课题、陪伴者风格、现有资源、本季新数据。优势必须使用具体场景，避免空泛标签。
- 使用 `### 观察线 1：名称` 至 `### 观察线 3—5：名称`。每条包含“观察什么、已有证据、本季度观察问题”。
- 使用 `### 环境 1：名称` 至 `### 环境 3—5：名称`。每个环境包含“为什么、怎么做、与观察线的连接、忙碌日最低版本”。
- 专属互动只为资料中真实存在且愿意参与的重要陪伴者设计；写具体活动、频率、孩子可以拒绝的方式和关联观察线。不默认父亲或任何家庭角色存在。
- “本计划观察指标”用绿灯、黄灯、红灯描述：绿灯为继续保持，黄灯为先微调，红灯为停止或联系当地合格专业人员。它不是通用月龄筛查表，不写“正常、超前、落后”。
- 家长话术使用“场景、可以说、避免说”三列，共 5—8 条；涉及接送、见面或照护时，只写家庭已确认的安排。
- 避坑提醒写 3—5 条，每条包含“风险、为何与本家庭有关、替代做法”。
- 每月复盘三问分别检查孩子体验、实际变化和家庭负担。
- 下季度展望只写条件性预判，以季度末新资料为准。
