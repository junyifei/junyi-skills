# 36—71 月龄聚焦方向决策方法

## 1. 先排除

- 未处理的即时安全风险；
- 需要专业判断的医学、心理、发育或教育诉求；
- 缺少有效全年底座、季度开始月龄或 `status: ready` 的季度更新；
- 缺少孩子最低限度的直接表达；
- 目标依赖尚未确认的入园、课程、照护或预算安排。

## 2. 形成候选方向

候选只能来自全年支持重点、本季 `Qxx`、家庭真实生活目标和孩子直接表达。标记发展萌芽、持续兴趣、外部节点、关系或生活卡点；每个候选至少绑定一条证据。

## 3. 按顺序比较

1. 孩子是否持续表现出兴趣、动力、拒绝或困难；
2. 是否处于萌芽到突破的窗口；
3. 是否存在升班、入学、照护变化等季度节点；
4. 证据是否跨时间、跨情境或多来源；
5. 成人、环境和流程是否能在 90 天内改变；
6. 家庭投入是否可持续；
7. 暂不处理是否存在明确现实代价。

不机械打分。选 1—2 个聚焦方向和 1—2 个自然维持方向。每份报告都解释为什么选这些、为什么其他方向本季度暂不加码。

## 4. 禁止作为聚焦结果

- 识字量、算术量、词汇量达到固定数字；
- 每天必须完成固定分钟数的学科练习；
- 为入学提前刷题或训练坐姿、服从、专注时长；
- 仅因同龄比较、家长焦虑或一次表现而形成的目标。

## 5. 负担与容错

- 每个聚焦方向提供 3—5 个游戏或生活建议，家长按情境选择，不要求全做；
- 已有稳定游戏、共读、运动和照护习惯写成自然维持；
- 生日、升班、旅行等一次性节点写条件性调整；
- 忙碌周使用每个方向的最低版本；
- 两周无变化先检查任务是否太抽象、步骤是否太多、成人是否一致、场景是否合适；
- 孩子压力、拒绝或家庭冲突扩大时，暂停或降低强度。
